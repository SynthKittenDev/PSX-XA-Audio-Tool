DESCRIPTION:

This program was created to modify songs in PS1 games (ex: Gran Turismo 2), but is usable in other PS1 games.

LIMITATIONS:

Aim for songs to be the same length as the original track. Longer will be cut off. Shorter will add silence to the end of your song. I recommend fading out the song just before the cutoff. The length of the original track is visible in the program.

INSTRUCTIONS:

1. Install latest Python 3 ( https://www.python.org/downloads )

2. Install requirements.txt ( pip install -r requirements.txt )

3. Run/Open PSX_XA_Audio_Tool.py

4. Open your BIN or XA file ( This tutorial will cover BIN, XA is for advanced users )

5. Select a track to replace ( Track 1-6 are the six race songs used in GT2 )

6. Select your 16-Bit WAV file, you can convert your songs from mp3 or other formats to WAV easily using Audacity ( https://www.audacityteam.org/download )

7. Choose "Replace & Save As..." and save your new BIN file under a new name

If you have any issues or questions, contact me below.
